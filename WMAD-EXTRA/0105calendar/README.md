# table tag

### table

### tr
table row

### th
table header

### td
table data


## horizon
```js
<table>
  <tr>
    <th> id </th>
    <th> student </th>
  </tr>
  <tr>
    <td> 1 </td>
    <td> sae </td>
  </tr>
  <tr>
    <td> 2 </td>
    <td> daniel </td>
  </tr>
</table>
```
